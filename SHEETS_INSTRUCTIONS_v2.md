### Маппинг "источник в форме → колонка стримера"

Стример выбирает в форме источник (например "🟡 Подписка / донат 150₽"), а формулы должны попадать в правильную колонку (зависит и от стримера, и от источника).

Используется такая логика в формулах: **выбор колонки = тип_источник_стримера**.
Пример: стример "PariPariKai" + источник "🟡 Подписка / донат 150₽" → колонка `gold_subs_donates_ppk`.

### Формулы для каждой колонки B–AQ

Все 45 колонок-источников (от B до AQ включительно) заполняются формулой SUMIFS из листа `FormResponses`.

Структура `FormResponses` (созданная Google Form):
- A: Timestamp
- B: Стример (PariPariKai / Ilyich116 / d1psy_off)
- C: Ник зрителя
- D: Источник билетов (одно из значений выпадающего списка, начинается с 🟡 или ⚪)
- E: Количество билетов

**Формула для B2** (`gold_subs_donates_ppk`):
```
=IFERROR(SUMIFS(FormResponses!E:E,
                FormResponses!B:B, "PariPariKai",
                FormResponses!C:C, A2,
                FormResponses!D:D, "🟡 Подписка / донат 150₽"), 0)
```

**Для всех остальных колонок B–AQ — та же формула, меняются два аргумента:**
- Имя стримера в `FormResponses!B:B` (PariPariKai / Ilyich116 / d1psy_off)
- Название источника в `FormResponses!D:D` (полная строка с эмодзи)

### Формулы для итоговых колонок AR–BA

**AR (`gold_ppk`)** — сумма золотых билетов от ПариПариКая по всем источникам:
```
=B2+E2+H2+K2+N2+Q2+T2+W2
```

**AS (`gold_ilyich`)**:
```
=C2+F2+I2+L2+O2+R2+U2+X2
```

**AT (`gold_dipsy`)**:
```
=D2+G2+J2+M2+P2+S2+V2+Y2
```

**AU (`silver_ppk`)** — сумма серебряных от ППК, **с применением капа 100**:
```
=MIN(100, Z2+AC2+AF2+AI2+AL2+AO2)
```

**AV (`silver_ilyich`)** — с капом 100:
```
=MIN(100, AA2+AD2+AG2+AJ2+AM2+AP2)
```

**AW (`silver_dipsy`)** — с капом 100:
```
=MIN(100, AB2+AE2+AH2+AK2+AN2+AQ2)
```

**AX (`gold_total`)** — общая сумма золотых от всех стримеров:
```
=AR2+AS2+AT2
```

**AY (`silver_total`)** — общая сумма серебряных (уже с применёнными капами):
```
=AU2+AV2+AW2
```

**AZ (`tickets`)** — ИТОГО, это видит колесо фортуны:
```
=AX2+AY2
```

**BA (`avatar_url`)** — подтягивание из листа Viewers:
```
=IFERROR(VLOOKUP(A2, Viewers!A:B, 2, FALSE), "")
```

### Заполнение формул

Все формулы (от B до BA) — **протянуть вниз на 500 строк** (или использовать ARRAYFORMULA эквивалент).

Колонка A (`nick`) — должна **автоматически подтягиваться из листа Viewers**, чтобы стримерам не приходилось вручную добавлять ники. Используется формула в A2:
```
=IFERROR(SORT(UNIQUE(FILTER(Viewers!A2:A, Viewers!A2:A<>""))), "")
```

Это значит: список ников зрителей в `Tickets` берётся из `Viewers` автоматически, отсортирован по алфавиту, без дубликатов.
